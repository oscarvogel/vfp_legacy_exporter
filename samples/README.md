# Samples

Esta carpeta debe contener únicamente ejemplos ficticios o sanitizados.

No subir:

- sistemas completos de clientes;
- datos comerciales reales;
- CUIT, claves, rutas internas o credenciales;
- DBF/FPT reales de producción;
- formularios con información sensible.

Uso recomendado:

- crear formularios mínimos de prueba;
- crear clases VCX ficticias;
- crear reportes simples sin datos reales;
- usar estos archivos para validar el exportador.
